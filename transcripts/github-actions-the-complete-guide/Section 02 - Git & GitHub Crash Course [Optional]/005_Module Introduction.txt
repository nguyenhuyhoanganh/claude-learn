# Module Introduction

This course section is optional because this course section is a Git and GitHub crash course, which you should take if you are not really familiar with Git and GitHub. Now, the course is, of course, about GitHub actions, but nonetheless, basic Git and GitHub knowledge is strongly recommended to fully understand how GitHub actions works and what you can do with that service. Therefore, in this course section, you will learn how exactly you can work with Git, how you can install it, and create new Git repositories and what the key Git commands are. You will of course also learn how to work with GitHub which is that cloud service that you can use in conjunction with Git, and you will learn how to create and use GitHub repositories. And, of course, you will also learn how to combine Git with GitHub, so local code management with cloud code management. That's what this section is about. As mentioned, definitely skip it if you do have that basic Git or GitHub knowledge. In that case, you won't need it. It's not a deep dive section, but otherwise, continue to make sure that we're all on the same page.
