# Module Introduction

In this course section, we will explore how you can use containers with GitHub Actions, and for that I'll also provide a very brief refresher on what containers are in the first place and why you might want to use these container things. Though you will get the most out of this section if you do have some knowledge about containers already. Otherwise, you can also skip this section. But it will reintroduce you to the concept and we will then learn how we can utilize containers for running our jobs in containers instead of just the runner machines, which we can specify. And we will explore a concept called Service Containers that's offered by GitHub Actions which is also a concept that can help us in our automated workflows.
