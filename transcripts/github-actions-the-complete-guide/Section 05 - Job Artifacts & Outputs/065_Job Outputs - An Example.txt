And to make it available, we must create a so-called output, which is a separate thing in GitHub Actions, which is different from artifacts. An output or an output value is a simple value. So not a file but instead, a number or some text or anything like this. A simple value that should be produced by the job that should be accessible by other jobs. And to generate and share such a output, we must add a new key here at the top level of this job definition before we define the steps. And that's the outputs key.
