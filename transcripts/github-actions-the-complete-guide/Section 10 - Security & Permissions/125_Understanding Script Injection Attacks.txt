So if you are using user generated values like the title of an issue, it's strongly recommended that you first store them in an environment variable, before using those values anywhere in the commands you are executing. So script injection is one potential problem you should be aware of, and you can defend against it by using actions instead of using your own run commands, or using environment variables where you store user entered data before using it.
