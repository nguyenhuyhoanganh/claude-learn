# Understanding Partition Assignment

This will be a quick lecture. I wanted to share this information. Where is this code? Who is calculating this partition? Is this done by the Kafka server? Actually, no, it is not done by the Kafka server. It is done by the client library. In our application, we will be producing messages, events along with the key. Then we will be submitting the messages or providing the messages to the client library to send to Kafka. So that time the client library using the key, then using this code, it will be finding the partition. Then it will be submitting that information to the Kafka server saying that this message should go to this partition and all. The Kafka server, it will simply accept the message and it will add in the appropriate partition. This is how it works. There is also one option to skip this function. That is, instead of letting this client library to calculate the partition, we ourselves can provide the partition if we want. But mostly we do not do that, but there is an option for that as well.
