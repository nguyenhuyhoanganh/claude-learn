# Welcome to virtualization

Back in my college days, when I started learning Linux, I had to build my own lab. I couldn't afford multiple computers, so I began collecting used computer parts. CPU, Ram, hard disk monitor, whatever I could find piece by piece. I assembled them, installed Linux, connected them with an Ethernet cable, and started practicing Linux server and client. If you tried building five, 10 or 15 Machine Lab today, it could cost a fortune. Back then, I didn't even know virtual machines existed. But today, you can create your entire lab inside your computer quickly, easily, and without spending any money. Virtual machine is simply a computer inside your computer. You can create multiple VMs, install operating system, connect them together and practice just like real multi node environment. And that's exactly what we'll do in this section. First we'll create virtual machines manually and then we'll automate the whole process. This setup is essential as we move into Linux section of the course.
