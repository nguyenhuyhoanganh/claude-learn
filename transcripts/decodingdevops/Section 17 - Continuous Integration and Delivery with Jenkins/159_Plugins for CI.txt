# Plugins for CI

Okay, so these are the steps. We have set up Jenkins. We have done a Nexus setup, Sonarqube setup, and we have taken care of the security group rule. Now let's take care of plugins. So here are the plugins that we'll be using in a few upcoming lectures, Nexus plugin to integrate Nexus with Jenkins, same Sonarqube plugin to integrate Sonarqube server with Jenkins, Git plugin, which we already have, and the tool as well, Pipeline Maven integration plugin, so we can execute pipeline code by using Maven and BuildTimestamp plugin. Some plugins we already have. Some we are going to install. So go to Manage Jenkins and click on Plugins. Go to Available plugins. Search for a Nexus plugin, Nexus Artifact Uploader. Search for Sonarqube plugin. Sonarqube. Sonarqube Scanner. Search for Maven, Pipeline Maven Integration plugin. Search for a timestamp. Use this text, Timestamp. See Build Timestamp. If it's already installed, check it in the Installed plugin section. If it's not installed, then you should see here in the Available plugin section for a check mark there also. Once you have checked all this plugin, then click on Install. It's going to take a moment to install. Once it is installed, join me in the next lecture.
