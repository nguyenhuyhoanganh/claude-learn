# Importing JSON Data with Logstash

JSON is a popular format for exchanging structured data as text between clients and servers, and vice versa. JSON, which stands for JavaScript Object Notation, came from JavaScript, but has since become so popular that it's supported by most major languages and even some databases. Properly getting your JSON data into Elasticsearch enables its most powerful features, as Elasticsearch is a JSON based search and analytics engine. In this lesson, we'll cover some basic concepts about JSON and then see how the Logstash JSON filter plugin parses JSON into structured Logstash events to be ingested by Elasticsearch. So let's get right to it. Before we jump into Logstash JSON filter, let's first lay down a few fundamental points about how JSON works. Here are some JSON basic concepts. JSON data is structured in key value pairs and ordered lists. Keys and values are separated by colons. Double quotes around each key like a name and each value. Each key value pair is separated by a comma, and the file type used is JSON. Booleans can be true or false. Values. Strings are Unicode characters that are wrapped in double quotes. Numbers are numeric characters without quotes. These must be an integer or floating point. Arrays are ordered lists with zero or more values that are wrapped in square brackets. Arrays can also include other JSON objects, and JSON objects are written as a collection of key value pairs and wrapped in curly brackets, and null represents an empty value. Let's take a look at an example JSON file. There are five JSON documents in the log, and each event is separated by a new line. This is the example log we'll be working with in the hands on section of this lesson. Logstash includes a JSON filter plugin as part of its default plugins. The plugin parses JSON data and creates corresponding data structure in Logstash event. Without this parsing step, the JSON data would be ingested into Elasticsearch as a single line of text, which would limit our options for searching and analysis. In this lesson, we will see how this Logstash JSON filter is used and some of its special use cases. So let's dive in and play with it.
