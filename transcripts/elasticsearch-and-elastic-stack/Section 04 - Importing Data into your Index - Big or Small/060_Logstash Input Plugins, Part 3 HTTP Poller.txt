Now let's cover the HTTP polar plugin. The HTTP polar plugin is a very useful input plugin that comes with logs dash. As the name indicates, this plugin can be used to pull HTTP endpoints periodically and let us store the response in Elasticsearch. This can be helpful if we have health APIs exposed for our applications and need periodic status monitoring of them. Also, we can use it to collect the status like weather, match details, etc. periodically. In our case, let's look at pulling two HTTP APIs for the purpose of understanding the plug in.
