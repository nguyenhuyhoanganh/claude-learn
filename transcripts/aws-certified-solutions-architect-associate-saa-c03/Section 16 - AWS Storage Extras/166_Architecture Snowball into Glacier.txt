# Architecture: Snowball into Glacier

So just a very short lecture around a scenario that can come up in the exam. And the idea is that you want to have Snowball import data directly into Glacier, but Snowball cannot import data directly into Glacier. The solution for you is to use Amazon S3 first, and then you would create a lifecycle policy to transition these objects into Amazon Glacier. So once you know, it is very simple. Snowball imports the data into Amazon S3 and thanks to an S3 lifecycle policy, the data is transitioned into Amazon Glacier. That is something to remember for the exam.
