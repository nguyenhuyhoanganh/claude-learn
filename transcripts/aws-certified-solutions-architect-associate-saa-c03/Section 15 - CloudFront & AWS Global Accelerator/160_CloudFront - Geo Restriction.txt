# CloudFront - Geo Restriction

Just a short lecture around CloudFront geo restriction. So you can restrict who can access your distribution based on the country where they try to access the distribution from. So you can set up an allow list to define a list of approved countries, or you can set up a block list to set up a list of banned countries. Now the country is determined by using a third party geo IP database to match the IP of the user to the country it belongs to.
