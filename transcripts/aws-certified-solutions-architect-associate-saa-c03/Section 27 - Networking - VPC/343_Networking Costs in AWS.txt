So now let's talk about how we can optimize our networking costs by making some smart architectural decisions. So egress traffic is outbound traffic. That means from AWS to the outside. An ingress traffic is inbound traffic. That means from the outside to AWS which is typically free. So sending data to AWS is usually free, but taking data outside of AWS, you have to pay. So your goal is going to be to try to keep as much internet traffic as possible within AWS to minimize the costs.
