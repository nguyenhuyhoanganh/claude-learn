System with Snap can install Helm using the snap install helm command. Use the classic option to install a more relaxed sandbox that gives the app a bit more access to the whole system. So rather than strictly isolating it to its separate environment, this way, Helm can easily access the kubeconfig file in your home directory so it knows how to connect to our Kubernetes cluster. For APT bases systems, such as the BN or Ubuntu, follow the instructions to add key and sources list before installing Helm. And for PKG, I'll run the pkg install helm command.
