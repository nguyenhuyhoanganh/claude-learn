# Null Object - Example

All right, Now let's look at some real life example of null object design pattern. Now, unfortunately, there is no perfect example of this design pattern. However, the adapter classes that are present in the WTA event package in the Java Class library can be thought of an example of null object design pattern. The only reason that these classes are not perfect example of this pattern is because those classes are abstract classes, so we cannot create an object of that class directly. However, these classes do not have any abstract method, so creating a null object using those classes will be a very simple code. You will simply extend from that class and keep that class empty and you will be getting your own null object in the place. So here is a code that is taken from the mouse adapter class which is present in the Java dot event package and you will see that this class implements multiple interfaces that are expected by various UI components in the AWP package and you will see that as per our definition of null object design pattern, it has implemented those methods, but it is not taking any action in those methods. Okay, so all these methods are empty methods and this is exactly what Null object does. The only reason this is not a perfect example is because this class is marked as abstract. However, there are no abstract methods in this class. Okay, so you can think of these classes mouse adapter. There are various types of adapters in this package. You can think of those classes as examples or a variation of null object design pattern.
