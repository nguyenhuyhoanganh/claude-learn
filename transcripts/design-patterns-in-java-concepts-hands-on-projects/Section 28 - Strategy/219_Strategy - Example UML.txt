# Strategy - Example UML

All right, Now let's look at the UML diagram of the example that we are going to implement in Java in next video. So to begin with, we have here a print to file service. So this is going to be our context. So this is the class whose object is going to use our strategy objects. We have a client here which will be a main method as usual. And this print to file service has an operation or a method called as print orders. And client is going to give us a collection of orders and job of this print to file service is to print those orders. Now we are going to use a strategy design pattern here to print these orders in completely different ways. Okay. So this is to highlight that when we we are talking about algorithm, we are not talking about searching and sorting only. So to come back to your example, we have an interface here called as order printer. So this will be our strategy interface, which has a method called as print. And it is again going to expect a collection of orders to print. There will be two implementations that we're going to look at. The first implementation is detail printer. So this implementation of strategy is going to print all details of the orders that are present in that collection. And then we are going to have a summary printer. And this implementation or this algorithm is going to simply print the summary of each order that is present in that collection. So our client is going to create object of one of these classes and then configure our print to file service object with that strategy object, and then it's going to call print orders and we're going to see different results based on which algorithm is given to us by client.
