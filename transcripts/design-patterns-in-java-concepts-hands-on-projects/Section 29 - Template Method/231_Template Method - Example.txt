# Template Method - Example

All right, Now let's look at some real life examples of template method design. Pattern and application of this design pattern can be found in the collections framework, which is part of Java class library and specifically the abstract map and abstract set classes. These classes have many non abstract methods which use or make use of other abstract methods that are defined in those classes to define some algorithm. So these are the examples of template method. So here is a code that is taken from the abstract set class and we have a method here called as remove all. Now you will see that this method is depending or calling on the size method which is defined in this class or its parent class as an abstract method. All right, so this remove all method is an example of template method, which is using other methods that are defined in the same class to define an algorithm. All right. So this is an example, real life example of template method design pattern.
