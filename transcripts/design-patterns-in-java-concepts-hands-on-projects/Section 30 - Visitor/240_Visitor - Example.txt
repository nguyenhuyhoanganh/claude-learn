# Visitor - Example

All right. Now let's look at some examples of visitor design pattern. So one example of this pattern can be found in the Dom Forge library that is used while doing XML parsing or using the XSLT processing. So this library has an interface called as a Visitor and a sample implementation called as Visitor support. So using these by implementing these interfaces and classes, we can define our own visitor and we can visit each type of element in an XML that is being processed by this particular library. Now another example of this pattern can be found in the new I O package of Java. And the interface that we are interested in is called as file visitor interface, and there is a sample implementation which is provided in the same package which is simple file visitor class. So here is a sample example where we are using the simple file visitor so we can define a class which extends from simple file visitor and it has a method called as visit file. Okay. So when we are working with file system and we have this visitor, then we can visit a directory and all the files that are present in that directory using this visitor. Once we visit a particular file in a directory, we can take some action on that file. So we can print either the file information, we can delete that particular file or any kind of processing can be done. So this is an example of visitor design pattern and the file visitor interface in Java's new package can be used to visit each file in a directory and take some action on those files.
