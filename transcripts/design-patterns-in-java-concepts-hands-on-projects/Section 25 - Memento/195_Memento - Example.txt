Then we had the originator, which is the object which gives us mementos, and then we have a caretaker rule which stores these mementos and can restore our original originator state in swing. The Undo Manager class is the caretaker rule, so this class will store implementations or objects of undoable edit So undoable edit implementations are mementos. The originator in these particular classes is the document implementations. So the document document class will actually serve as the originator for our class. Now in order to get the undo functionality or in order to use the momentum, we need to access the document because remember, document actually is the originator.
