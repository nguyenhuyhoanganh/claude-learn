# Command - Example

All right, Now let's look at some real world examples of command design pattern. Now we'll see that there are many classes that are part of swing which follow this command design pattern. However, we are going to look at some other classes where this design pattern is used. An example of command pattern is actually the runnable interface, which is part of Java class library because we create the class which implements our runnable interface, which has defined a single method called as run. And inside this method we will call an operation on the receiver or whatever functionality that we want to provide, and we can send this object to other parts of our application where that object can be executed on a different thread. So runnable is also an example of command design pattern. Another example of this pattern is the action class in the Struts framework. So in this particular framework we configure each URL or we map each URL to a particular action class. And in that particular mapping we can also provide the actual method that needs to be called. And that way we have our command. So whenever a request comes on that particular URL, our action class will be created and that particular method will be executed. So Action Class Struts framework is also an example of command design pattern.
