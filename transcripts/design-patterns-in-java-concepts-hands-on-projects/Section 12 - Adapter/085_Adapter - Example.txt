# Adapter - Example

All right. Now let's look at an example of an adapter pattern in the Java class library. And the best examples of that are the input stream reader and output stream writer classes, which are part of Java's i o package. And these classes are examples of object adapter, so these classes will adapt our existing input stream or Outstream output stream objects to reader and writer interfaces which are also part of the Java IO package. So let's look at the code of the input stream reader. Now remember, this code is taken from the dart jar file and you will see that we have a class here called as Input Stream Reader. And in the constructor we are giving an existing input stream object to this adapter. Now we will see that this class is extending from the reader class and this is how it is able to adapt to the reader interface. So while providing the operation implementations for the operations that are defined in the reader, you will see that our input stream reader is going to use stream decoder because sometimes the adapters have to do some additional work in order to adapt to the new interface. So using the stream decoder, it will delegate the operations to the underlying input stream object. All right, so input stream reader and output stream writer are examples of object adapters in Java class library.
