# Decorator - Example

All right. Now let's look at an example of decorator design pattern in Java class library. And as you might have already known, the Java package is a really great example of this decorator pattern. And in this particular example, we're going to look at buffered output stream class, and this particular class can decorate any output stream object and it adds the buffering capability to that particular output stream. And a benefit of this buffering is that it improves the disk i o performance because it reduces the number of blocking operations, blocking i o operations that are done. So let's look at the UML diagram of this particular these particular classes. So first of all, we have the output stream and you can think of output stream as the abstract component in our decorator. Then we have file output stream, which is a concrete component, and then we have a class called as filter output stream, which is our decorator. The buffered output stream is actually extending from our filter output stream, and this buffered output stream is our decorator. So when we are working with buffer output stream, we give that output stream an object of file output stream or any child class of output stream. And this particular decorator will add the buffering capability to that output stream. All right, So there are many other classes in the Java IO package that you can look and you will find that they are great examples of decorator where each class adds a little bit of functionality on top of what is provided by the original object.
