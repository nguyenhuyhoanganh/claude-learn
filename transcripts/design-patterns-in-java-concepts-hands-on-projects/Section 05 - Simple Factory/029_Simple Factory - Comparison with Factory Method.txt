# Simple Factory - Comparison with Factory Method

All right. Now let's look at some of the differences of simple factory with factory method. So Simple Factory is simply an encapsulation of our object creation logic. So we take our instantiation logic and we put that in a simple static method in a separate class factory. Method, on the other hand, is more useful if we want to delegate the object creation decision to subclasses. Now Simple Factory knows about all the classes whose objects it can create. Factory methods. On the other hand, doesn't know about all the product subclasses which can be created because the decision about which class to instantiate is delegated to subclasses of factory method. All right. So these are the differences between simple factory and factory method.
