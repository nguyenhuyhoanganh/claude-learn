Those would be any time that you have any append only or prepared only data. So if it turns out that you just keep on adding in additional readings over time or additional records over time to some set of records, well, that might be a good use case for a list. Another good use case is any time that we need to only fetch the last or first couple records of some collection of records. Examples of this might be if we are trying to show social media posts and we only want to see the latest five posts from someone.
