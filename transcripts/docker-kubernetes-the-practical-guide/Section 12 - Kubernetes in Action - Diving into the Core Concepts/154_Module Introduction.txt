# Module Introduction

In the previous course section, I introduced you to Kubernetes, and I outlined the big picture, the general architecture, and well, I gave you some terms and some core concepts, which you will see from this module as well. Because in this course module here, we are going to dive into Kubernetes, and actually get our hands dirty working with it. And for that, we will first of all, learn how to set up Kubernetes and a Kubernetes testing environment, so that we can start playing around with Kubernetes. We'll then learn how to create Kubernetes deployments, and how to work with Kubernetes objects, what objects are, and which kinds of objects Kubernetes knows. And of course, we're going to do that at concrete examples. So, that by the end of this module, we actually deployed our first basic application, with help of Kubernetes. For the moment, locally on this testing environment, on our local host machine, but of course, with that, we'll then have all the knowledge we need, to also apply the same Kubernetes configuration, at a larger scale on any Cloud provider, or any remote machines of our choice.
