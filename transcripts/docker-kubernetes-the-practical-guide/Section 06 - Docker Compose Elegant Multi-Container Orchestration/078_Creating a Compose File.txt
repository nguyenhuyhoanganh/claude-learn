Now, in such a docker compose yamol file, you describe your multi container environment, your project setup, and you always start by defining a version here. Now you should get auto completion here, by the way, since we installed the docker extension, make sure you have that Docker extension installed in visual studio code and with that you will get all the completion here, which is very helpful.
