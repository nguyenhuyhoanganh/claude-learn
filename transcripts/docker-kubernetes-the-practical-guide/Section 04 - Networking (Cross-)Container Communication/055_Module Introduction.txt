# Module Introduction

Now that we've covered a lot of basics about images and containers and now that we all also know how we can persist data inside of a container and how we can work with volumes, we'll dive into a never key concept which you will need in most applications you're going to dockerize and that will be networks and how you can use networks from inside your containers. Now with networks here, i really mean how you can connect multiple containers, how you can let them talk to each other but also how you could connect an application running in a container to your local host machine. And for example, send HTTP requests to some other service running on your machine. And also how you can reach out to the world wide web from inside your container. Specifically in this module, we are going to explore which kinds of networks we have or where you might want to send requests to from inside your containers. And we're going to explore how we can connect multiple containers so that they can interact with each other.
