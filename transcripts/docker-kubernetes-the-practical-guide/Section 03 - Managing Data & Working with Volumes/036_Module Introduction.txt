# Module Introduction

Up to this point in the course, we learn a lot about images and containers, And we got a very solid foundation for diving deeper into Docker and deeper into working with these concepts. Now in this module, we are going to explore how we can manage our data inside of images and containers. This might sound strange at first, because up to this point, we already had data in images and containers, right? We already had our code in there for example. Well, turns out there are different kinds of data, and up to this point, we're only able to manage one kind of data in our images and containers, and we'll face problems with the average kinds. We'll then, throughout this module, dive into how images and containers can manage data in different ways and how you can connect to different folders and much more and we'll specifically dive into a concept called volumes, which is built into Docker. And towards the end of this module, we're also going to explore a kind of related concept. We're going to explore arguments and environment variables and see how we can use them in Docker images and containers and why we might wanna use those things.
