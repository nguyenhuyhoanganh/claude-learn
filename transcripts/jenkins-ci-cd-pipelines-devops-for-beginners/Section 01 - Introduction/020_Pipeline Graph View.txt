# Pipeline Graph View

Apart from the stage view that we can see when we are on the status page of a pipeline. There's also another view where we can see the stages of the pipeline. This is accessible here from the menu under stages. This is a bit of a newer development in Jenkins, and this is actually called a pipeline graph view. Here you're going to see your entire pipeline. And if you click on individual stages you're going to get details about the different actions that happen in that particular stage. For example, here there was a message that was printed. Here is a shell script that has been executed. And you can of course expand these and see exactly what has happened inside that. Same goes for a different stage. We have here the test stage. I'm going to see here the shell script that has been executed. And it just offers a different overview altogether. We will not be using this during the course, but I think it's a good idea to know that it exists. There are different options here. You can start the build directly. You can view the console. You can also configure the job. So you're not lost in this new interface here which this view brings. All right. So I guess this is all you need to know about the pipeline graph view.
