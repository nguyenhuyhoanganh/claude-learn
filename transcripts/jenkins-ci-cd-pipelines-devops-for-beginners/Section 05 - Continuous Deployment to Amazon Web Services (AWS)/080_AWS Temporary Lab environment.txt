This lecture is specifically for those who have access to labs, which may not include everyone. So if you don't have access to labs, feel free to skip ahead. You won't be needing this throughout the section. So the first step is to open the AWS console, which can be done by clicking this button, which says here open workspace. Please note that this is a temporary lab environment that will automatically terminate after 60 minutes. There's a timer at the bottom indicating the remaining time. If you're browsing around the AWS console, you're going to notice that it will look slightly different from what you see in the rest of the section. But don't worry, the core concepts remain the same. You may want to create a new workspace once the hands-on portion of the section begins to ensure you have enough time to practice. In my experience, the AWS environment that you see will stay accessible slightly longer than the indicated 60 minutes, but I can guarantee that will always be the case. The last thing I wanted to show you is how to get AWS credentials. Now later in this section, you will need AWS credentials to automate a deployment from Jenkins to AWS. And since you're using a lab, you won't be able to create these credentials manually as I demonstrate in the course, so you'll need to skip that part. You're going to see here in the interface, another button called CLI credentials. Now if you click on this, what you're going to see here is the AWS CLI credentials that you're going to need later in the course. Now note that these are temporary AWS credentials, which are actually easy to spot because they start with this Asia prefix here. Now in contrast, long last YAML user credentials usually start with A-K-I-A. Now it doesn't matter so much. The main point that I wanted to bring here is that you're going to get a set of three values, the AWS access key ID, the secret access key, and also a session token. Now because these credentials are generated by AWS STS, typically by assuming a role or starting a temporary session, you will also need to provide the whole bunch. Now later in the section, you will need to define and provide all three values, not just the two values that I'm showing in the course. So whenever you're trying to authenticate with these credentials, you need to provide all three of them. So just keep that in mind.
