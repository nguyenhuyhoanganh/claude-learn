# Creating a GitHub account

So far we have stored the pipeline configuration directly in Jenkins. This is okay if you want to try something out, and we don't want to go through the entire process. However, if we want to truly follow the pipeline as code principle, we need to store this pipeline code in a code versioning system like git, which is probably the most popular solution. To work with git and use a git repository. We will use a very popular git hosting platform called GitHub. So I'm here at github.com. And here at GitHub you can create an account for free. For the next part of the course you will need an account with github.com. It is absolutely free to register for one. We will use GitHub to store our project code repository as well as the Jenkins pipeline file. So go ahead, sign up for an account, follow all the steps and I will see you in the upcoming lecture.
